This project contains a climate dashboard developed using ERA5 monthly averaged data and IMD data.

Folder structure:

app.py – main Flask application file
templates/index.html – dashboard interface file

How to run:

Open the project folder in VS Code or any Python editor.
Make sure the required Python libraries are installed.
Run the app.py file.
After running, a local link will appear in the terminal.
Open that link in a browser to view the dashboard.

Note:
The dashboard displays the impact of extreme temperature and precipitation events on soil moisture using ERA5 and IMD datasets.
